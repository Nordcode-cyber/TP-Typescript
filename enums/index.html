Trabajando en Typescript con Enums

Trabajar con enums en TypeScript es muy similar a trabajar con ellos en otros lenguajes de programación. Los enums en TypeScript son una forma conveniente de definir un conjunto de valores constantes con nombres descriptivos, lo que hace que el código sea más fácil de entender y mantener. Una vez que se ha definido un enum, se puede utilizar para declarar variables y asignarles uno de los valores del enum.

function obtenerTextoTamaño(tamaño: Tamaño): string { switch (tamaño) { case Tamaño.Pequeño: return "S"; case Tamaño.Mediano: return "M"; case Tamaño.Grande: return "L"; case Tamaño.ExtraGrande: return "XL"; default: throw new Error("Tamaño inválido"); } } let tamañoTexto: string = obtenerTextoTamaño(Tamaño.Grande); console.log(tamañoTexto); // "L"
